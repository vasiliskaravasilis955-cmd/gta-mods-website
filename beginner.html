Story Mode Modding — Safe Essentials

This guide is for offline Story Mode modding. Follow each step carefully, keep backups, and do not use mods in GTA Online.

Requirements

  • Backup first: Copy your GTA V folder or at least the game executable and any files you change.
  • Use a mods folder: Install replacements inside mods so the original game files remain untouched.
  • Script Hook V: Required for many script mods. Place the ASI loader and ScriptHookV.dll into the GTA V root folder.
  • Script Hook V .NET: Required for .NET-based script mods. Install files into the GTA V root and create a scripts folder for .dll mods.

Common support files

  • NativeUI: Used by many script mods for menus and interfaces.
  • iFruitAddon2: Required by some vehicle and trainer mods.
  • Enable All Interiors: An interior unlock script for story mode scenarios.

Recommended workflow

  1. Install OpenIV and enable the mods folder.
  2. Install Script Hook V, then Script Hook V .NET if needed.
  3. Place scripts in scripts and assets inside mods as instructed.
  4. Test each mod offline in Story Mode before adding another.
  5. Keep a clean backup copy of your original game files.
